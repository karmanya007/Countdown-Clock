# Countdown Clock

Simple Countdown Clock

Installation
-------

### Requirements

You will need [Nodejs](https://nodejs.org/en/) version >= 12.12.2 and npm version >= 6.14.4

### Setup

1. Clone this repository.

   ```sh
   git clone https://github.com/karmanya007/Countdown-Clock.git
   ```
2. Open index.html
